Guiding PrinciplesEvery architecture decision in this document is governed by three rules:Single source of truth — Logic lives in one place. Business rules live in Cloud Functions, prompts live in Remote Config, data lives in Firestore.2933699-165861790700221538Language-pair agnostic from day one — "Sinhala" is never a hardcoded string. It is always a config variable ( si → en ). Adding Tamil → English later is a config change, not a code change.Server enforces, client displays — The server is the authority on usage limits, subscription status, and AI responses. The client's only job is to capture audio, play audio, and render UI.2. Monorepo StructureSimple flat folder structure.
